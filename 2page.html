AI와 관련된 직업을 갖기 위해 해야 하는 것

인공지능과 컴퓨터와 관련된 직업이 여러 가지라는 것과 데이터 사이언티스트를 하려면 수리 통계 지식, 프로그래밍 능력, 도메인 문제 해결 능력이 필요하고, ML 엔지니어를 하기 위해서는 수리 통계 지식, 백엔드 개발 역량이 필요하다는 것과, 데이터 엔지니어는 분산 시스템 개발 역량, 인프라 개발 역량, 백엔드 개발 역량이 필요하고, 데이터 분석가는 수리 통계 지식, 도메인 문제 해결 능력, 리포트 작성 및 시각화, 커뮤니케이션 역량이 필요하며, AI 연구자는 대학원 수준의 연구 경력, 수리 통계 지식, 프로그래밍 능력이 필요하다는 것을 알게 되었습니다.

과거의 인공지능과 지금의 인공 지능의 차이

예전에는 인공지능 등을 개발할 때 일일이 코드를 찾아서 적었지만 요즘에는 챗 gpt 등에서 검색하면 코드가 나온다는 사실이 놀라웠고, 예전과 달리 인공지능을 이용한 기술이 점점 더 많아지고 있다는 것을 알게 되었습니다.

나의 진로의 미래

인공지능 기술은 점점 더 다양해지고 그것이 단순 수작업을 요하는 직업 뿐만 아니라 인공지능을 사용하는 직업 조차도 대체될 수 있다는 것을 알게 되었고, 나는 나의 진로인 컴퓨터나 인공지능과 관련되어있는 직업 중 인공지능이 대체할 수 없는 직업에 흥미를 가지며 탐구하며 그것을 내 직업으로 삼아야겠다고 생각했습니다.
